# 演化系统综述

### 系统演化

差异普遍存在，能释放能量的差异称为势差。系统是世界变化过程中的一段，以势差作为系统边界，势差除了大小还带有方向，指向内部的势差决定系统的存续，指向外部的势差决定系统间的相互作用，外部势差决定了系统演化方向，系统总是需要适应外部同时维持自身稳定。系统的存在体现为势差疏解，试图消除差异。系统的任意状态显现出结构，能量流转过程显现为模式，势差会释放能量驱动模式展开，能量是系统内部赋权，它决定了能量流转的模式，每次流转都会有能量从系统内部流转到外部。同一套系统载体可以承载多种类型的势差。

谈论势差需要分清势差方向，是对内部的还是外部的，不同方向的势差疏解方式不同。势差在得不到有效疏解的情况下会进行量的积累，当超过某一临界点，系统就会发生重组，形成质变。

系统间的相互作用，外部系统作用系统本身启动内部的疏解模式，同时影响外部系统，外部系统的变化如果被本系统接收形成反馈式模式闭环，就会形成演化系统。系统调节发生在每次模式调用之后及下一次模式调用之前，它不能进行精确的预调节，只能进行动态逼近。

演化系统的形态和复杂度取决于其处理反馈的形式，如果只是机械处理则为机械系统，如果具备差异识别并能够以此为依据动态调节那就是智能系统。意识就是处理差异识别（隐含比较相同）的处理器，意识的复杂度与群体资源分配方式正相关，他们是迭代式演进的关系。蚂蚁群落不需要复杂的意识形态，人类社会则是必须，人类社会总是通过以不断创新的方式来疏解资源分配不均的现实。

基于差异识别的处理器如果将差异按照符号处理那就是形式化处理器，这里将任意智能的意识处理都认为是形式化处理只是有复杂度和处理机制不同。语言（本文中后续语言全部指人类语言）是意识处理的操作系统（语言行为与神经活动之间存在双向塑造关系），语言起源于差异识别，形成概念，概念形成网络，处理方式为网络激活和收敛，收敛即赋意，赋意是下一个网络激活的指引。意识活动总是按照指引展开。意义对于形式处理没有额外帮助，它要么是结果要么是中间产物，而具身体验伴随处理过程而产生，不是意义决定体验是意义与体验共同产生。

非符号化的处理（如直觉、技能、具身反应），本质上是有意识形式化处理的模式化产物，模式被差异处理器激活（比如示意）后展开。另外通过非符号化传递的感觉或感受（示意）因其信息承载能力有限不足以支撑复杂信息传递，但智能的进化定是以此为基础的演化，而在复杂信息中这些感觉感受就被符号化编码。

意识是物理基底的涌现，语言是意识的操作系统，逻辑是语言的一种表达方式，它的特点是保真性。

语言的遮蔽性，语言是观察者视角，它隐含了参考系，分辨率及时态，同时概念的内涵和外延也在指引上遮蔽概念的反面。连续的语言解读会进一步加剧其遮蔽性。

- 参考系，观察者的视角及其描述标准。
- 分辨率，描述对象的尺度，不同尺度见隔着涌现是不可通约的，所以描述问题不能在不同分辨率下反复切换，比如你不能用整体的概率去描述个体行为，人性是群体性它对个体的确定性无效。
- 时态，描述的对象是状态还是过程，状态和过程也是不可通约的，也不能在状态和过程间反复切换。比如爱情它不是个状态，它是个过程，如果说追求爱情，那不是站在那里那是需要行动的。

信息是指引网络，它直接操作意识激活和收敛网络，多模态信息也会最终通过语言被意识加工处理。语言同时也是信息的一种承载方式。意识总是在处理信息，信息的置信度是首要问题，但个体无法根本上独立确认信息的置信度，必须依赖他者，但是他者就是信源本身的话就应该警惕，人的注意力很容易被信息引导并随着信息的输入随波逐流，毫无察觉。

人通过有限算力的意识中介世界，目的是意识收敛出的未达状态，形成行动势差，驱动行动，过程中又会被意识识别对行动进行调节形成闭环。反复激活的网络会固化形成固定模式，固有模式也会通过情绪或者预设的处理钩子唤醒意识进行模式识别进而进行模式调整。又因为语言自身的遮蔽性及信息的置信度直接影响概念网络与势差网络的对齐，这也就决定了意识不可能在行动前精确的预知行动后的系统的变化，而只能不断的通过行动来趋近。追求方向比追求确定性更重要，答案总是在行动后。

### 形式化处理

网络收敛即明确网络中的链路，或者说任何概念皆可展开为节点路径。网络的收敛包括基于相同差异的整合叫做泛化；基于对复杂子网络的命名叫做标签化。路径泛化是对路径节点的泛化和路径简化。泛化概念仍具备可比较能力可以进行进一步的意识加工，标签化概念因其隐藏复杂路径导致无法进行再次比较操作而无法进行进一步加工。泛化概念可以进行形式化处理，而标签概念则几乎不可以。

回溯式封装是标签概念产生的重要方式。将个体行动的复杂过程简化为一个标签，遮蔽观察者认知网络中的复杂路径。个体网络标签概念。标签概念时常发生，因为简单高效，又因为遮蔽的路径不再被激活导致只剩下一个标签概念，其复杂网络被遗忘，导致回溯困难。这与具身状态描述一样，无法识别差异，直接进行标签化，比如疼痛，烦恼等。

指引是指示意识激活某个节点或进行节点比较。意识活动即随着指引展开并收敛概念网络的过程，节点即为意识关注点。从一个节点开始根据指引不断激活节点，并根据连接情感权重释放情感（即激活感），情感随之不断变化并提供情感指引。每次局部的网络收敛可作为后续的路径收敛的指引，此次收敛可作为临时节点，比如中途可能收敛出意图等。整个收敛过程意识会按照指引激活相关子网络，并在激活子网络中不断收敛或相互切换或相互引导。自动化指引包括节点连接指引，依赖记忆；情感指引，这是具身指引始终生效。情感可以抑制节点激活和节点比较，比如时间紧迫，抑制更多节点激活以及比较；负向情感倾向选择负向情感通路，正向情感倾向选择正向情感通路，这种路径选择性表现出情感对路径收敛的抑制性。

指引网络，是以基础指引——激活节点或节点比较为节点形成的认知网络，提供复杂的步骤化、程序化的指引，比如检查路径是否符合形式逻辑，通过指引检验路径节点以及逻辑关系是否一致来检验路径的逻辑合理性；因果评估，指引评估路径的因果性是否一致；路径整理指引，是对路径的泛化干预，查看节点类型，以及连接的合理性以及是否需要被泛化，特别在行动后的归因时整理，因为此时是具身体验；基于网络收敛结果的指引，根据某个网络收敛结果指引网络的激活或收敛或启动行动，比如回忆，去运动等。指引网络需要被激活才能使用。

重复激活路径可使路径强化，强化的路径会形成模式。在模式形成过程中，**情感从持续激活转变为边缘监控状态**——情感并未完全消失，而是退居背景，仅在路径遇到异常或需要重新评估时才被唤醒。这种‘情感背景化’使得常规运行低耗能，同时保留了必要的干预能力。模式化有两种形态：一种是技能化，路径压缩但保留情感钩子，使得异常情况能迅速唤醒意识进行干预；另一种是标签化，路径被完全隐藏，情感剥离，进入自动化运行。技能化是系统追求的高效状态，标签化则是需要警惕的认知陷阱。

### 系统存续

势差的存在是系统存在的根据，势差疏解能力决定了系统效率。对于人类社会而言，势差的可见性决定了系统的规模，而信息传播是激活势差的唯一形式。人的势差是需要激活且高度个性化的，被激活的才会被关注，被关注的才会驱动行动，行动后才会有事实效果，所以人的势差总是被意识激活是语言的，语言是差异的处理器，势差总是藏在人的关注点里，势差不需要被识别只需命名（注意，现在只有语言了别关注势差了，不要琢磨找势差了）。

系统内部对其元素有疏解的功能性约束及赋权，任意元素是势差网络的交叉点，它天然的要承担来自多重系统的功能性约束和赋权，于是次生系统便有了生发土壤，次生系统是在主系统中寄生的不是指向主势差疏解的内部子系统，它会将主系统的能量流转到系统外部，而不是下游。自锁次生系统，是系统元素依赖某系统存在，且其行动后反馈要求其继续保持存在，它维持自身也即维持系统存续，一旦形成模式就会稳定存在，这种次生系统需要关注，因为打破模式需要进行系统重构会付出可大可小的代价，这就是系统惯性。

如果将自锁分为乐观自锁和悲观自锁，乐观自锁是非负向反馈循环，悲观自锁就是负反馈循环，客观自锁可能遮蔽这个模式本身而忽略，一旦模式松动可能带来意外破坏，比如索取型模式，开始可能没被注意到，但是给与不够时就会带来抵抗。

系统元素更新机制是维持疏解效率的关键。任何系统都有生命周期，疏解效率会随着生命周期呈现不同的变化，系统为了维持疏解效率就必须有元素的更新机制，根本上说对于系统而言疏解效率是第一位选择标准因为这决定了系统的存续，处理不当系统崩溃。

任何系统都是在更大的系统中生发出来的，它总是嵌入在更大的系统中。系统的存续是更大系统对其疏解能力的选择，功能相似的子系统的存续不由彼此决定，最终都由大系统进行选择。

中国古代王朝的核心势差是让百姓活下来、活得好，活下去（这就是王权神器），这三个贯穿整个中国古代史。举个例子比如大明王朝，从朱元璋建立王朝起上面三个势差就是帝国主要疏解的内容，儒家建立和维持社会结构，采用士大夫来作为疏解系统，制定政策、组织生产、治理水患、收取税赋等，百姓在这3个势差稳定中得以存续。但是政权内部滋生次生系统用于疏解钱权势差，这个次生系统还是自锁的，它将流向百姓的能量返回来向上流动，百姓势差无法疏解，当突破势差则发动了农民起义使整个系统崩溃。

### 心智成本管理

人之所以为人的存续底线有生理和群体审视两种，主体性是个体演化的结果，而关键约束就是群体审视，这是收敛的底线约束。

世界不变，心不能根本的改变，对过去的念念不忘是当前的处境还不如过去，如果现在的处境远好于过去，过去已被事实遮蔽（行动遮蔽）。重要的不是自我说服，重要的是用实践改变现状。意义只是事后的叙事，是情感的慰藉。没有绝对的真理指明未来的路的具体展开，路只有走过才是路。

在系统中存续重要的是“安住当下并寻求突破”，而心智成本是存续的根本性成本，不妥善管理直接影响存续质量。一般理性可以大致分为目的理性用于收敛行动方向，另一个是工具理性用于抵达目的。目的理性是非常消耗算力的，工具理性则小很多，好的工具能让人直接进入心流状态。

安住当下意味着只需履行系统赋予的责任，承认责任履行责任是极低的心智消耗。责任生成目的（我该的那部分及提纯的我想的那部分），工具指向目的，目的确认责任。责任是系统性约束（注意它只对我这侧有效，公平没有约束效力，也就是它只约束我应该如何如何），被主体确认/承认的责任才会进入主体生存上下文才会生成目的，当客观不支持主体事先承认的预设时要主动废止。履行责任意味着对过去的维持。

“我想”的部分是突破的关键，但是并不是所有的“我想”都符合现实，心智成本管理的核心内容是如何提纯“我想”，情绪是心智负载的指示器，情绪也是唤醒理性的指引，这里还要关注同一个势差唤起情绪的频率和强度，这个反自锁的关键指标。

个体总是势差网络的交叉点需要同时处理多个势差问题，但是要区分哪个是主导势差，主要势差是那个如果不疏解，系统就会解体或质变的势差，次要的则是那些“虽然让人不适，但不至于立即危及存续”的势差，而且还必须注意势差的方向，是维持自身还是向外突破。优先处理主要势差。

心智成本管理本质就是如何展开有效的概念网络并使其得到有效收敛，方法就是进行有效提问和有限承认。

- 提问。对注意力(关注点)提问，目的是识别势差，识别其所在的系统以及其变化的方向。情绪是心智负载的指示器，情绪是一个非常好的关注点。有了关注点，恢复语言的遮蔽让势差显现，然后问为什么，观察其所在系统及系统的层级间的关系，分析系统的模式及其演化方向。情绪关注点一般都指向“我想”，通过展开“我想”来聚焦势差。
- 承认。承认算力/体力有限，承认事实与逻辑，承认不可以言说之事的存在，承认不可达，承认系统演化性。承认了就可以收敛网络了，网络收敛就会腾出算力。

上面实际是根据指引建立的一套元认知指引网络，目的是根据情况自动生成有效指引，随着有效指引展开网络而不是停下来或者随波逐流，然后根据情况进行有效的承认来收敛网络，给出方向投入事件。但是只是一个开始，它会在使用中自己生成完善的。元认知网络的初始运行需要消耗心智资源，但随着路径重复会逐步模式化，成为低耗能的自动化指引。在低算力状态下，可暂停元认知调用，仅依靠已固化的责任模式维持存续，或直接进入休息状态恢复算力。

上面的论述是写在行动前的，它强调有效导入实践，所以刻意规避情感问题，情感不应成为实践的主要参考或阻碍更应成为行动后的体验。行动后一定要注意反馈，这可能让自己陷入系统性自锁，比如学习一门课程，如果效果不佳那不一定是你的问题，如果被指引到是自己问题的话，你就会反复尝试，完成系统性自锁，这种情况如果频繁发生或强度越来越大一定要果断退出，工具好用则用，不好用则换。
